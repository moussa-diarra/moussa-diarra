## Featured projects

### [Active Directory Security Lab]
> Full attack & defense lab — Kerberoasting, Mimikatz, Event ID correlation, GPO hardening

Deployed a Windows AD environment from scratch in VirtualBox. Simulated real attacks (Pass-the-Hash, Kerberoasting), then built detections using advanced audit policies and Event ID 4625/4768 correlation. Concluded with a full GPO hardening pass and documented anomaly detection methodology.

`Active Directory` `PowerShell` `Mimikatz` `Windows Event Logs` `GPO` `Blue Team`

### [AD Security Audit Script]
> PowerShell automation to audit AD misconfigurations and weak account policies

Lists admin accounts, flags inactive users, and checks password policy compliance. Designed to run as a scheduled hardening check.
